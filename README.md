# LearnPlaywright2X

A comprehensive learning repository for Playwright automation and JavaScript fundamentals.

## Project Overview

This repository contains structured learning materials organized by chapters, covering:
- **Playwright Test Automation**: Basic concepts and advanced techniques
- **JavaScript Concepts**: Fundamentals including identifiers, literals, and ES6+ features
- **Development Tools**: VS Code shortcuts and best practices

## Repository Structure

```
LearnPlaywright2X/
├── Chapter_01_Basics/
│   ├── 01_Basic_First Code.js
│   └── 02_HotCode.js
├── Chapter_02_JavaScipt_Concepts/
│   ├── 05_JS_Basics
│   └── 06_identifier_Rules.js
├── Chapter_03_Identifiers_Literals/
│   └── 07_VS_Code_Keybaord_Shortcuts_windows
└── README.md
```

## Chapters

### Chapter 01: Basics
Introduction to basic Playwright and JavaScript concepts with foundational code examples.

### Chapter 02: JavaScript Concepts
Deep dive into JavaScript fundamentals including:
- JavaScript basics and syntax
- Identifier rules and best practices

### Chapter 03: Identifiers & Literals
Exploration of identifiers, literal values, and developer tools including VS Code keyboard shortcuts.

## Getting Started

1. Clone this repository
2. Ensure Node.js is installed
3. Install dependencies: `npm install`
4. Run example files: `node <path-to-file.js>`

## Prerequisites

- Node.js (v14 or higher)
- VS Code (recommended)
- Git

## Usage

Each chapter contains example files that can be executed independently:

```bash
node Chapter_01_Basics/01_Basic_First\ Code.js
node Chapter_02_JavaScipt_Concepts/06_identifier_Rules.js
```
